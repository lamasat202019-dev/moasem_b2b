# منصة مواسم B2B

منصة تجارة جملة ذكية (Web + Mobile) موجهة للتجار والموردين.

## المميزات الرئيسية
- تسجيل موردين وتجار
- متجر مستقل لكل مورد
- تخصيص الهوية (شعار – ألوان – تخطيط)
- سلة شراء وطلبات جملة
- محفظة رقمية (Wallet)
- لوحة تحكم احترافية
- تصميم عصري متجاوب مع الجوال

## التقنيات المقترحة
- Frontend Web: HTML / CSS / JavaScript
- Mobile App: Flutter
- Backend: Firebase / Supabase
- Hosting: GitHub Pages (مؤقتًا)
